Hexo与WordPress、Typecho的主观对比与评价

发表于 2021-08-18 更新于 2022-03-11 分类于 随笔 Waline： 本文字数： 378 阅读时长 ≈ 1 分钟

暑假闲来无事，就在家里试着用三种不同的博客框架部署个人博客。下面来对它们三者进行一些主观的对比与评价。

Hexo

优点

  • 轻量简洁
  • 可以使用Git + Github Pages/Vercel + Hexo的方法实现安全性高的完全白嫖
  • 对MarkDown的支持性好
  • 源码级高度自定义
  • 第三方主题质量高

缺点

  • Hexo站点难以部署到虚拟主机上
  • 需要先配置环境、掌握简单的MarkDown与Git知识，因此对零基础小白不友好
  • 静态网页维护起来困难
  • 只能用PC管理站点，用手机写文章是不可能了

WordPress

优点

  • 环境安装和对站点基础维护傻瓜式，操作简单
  • 海量的插件与主题支持
  • 官方更新速度快
  • 对移动设备的支援到位
  • 第三方主题质量高

缺点

  • 相比其他的博客框架更为臃肿

Typecho

优点

  • 轻巧高效，非常适合单纯地写文章
  • 简洁明了的后台管理系统
  • 完全的中文支持
  • 第三方主题质量高

缺点

  • 个人感觉编辑器不太好用
  • 定制功能较少
  • 插件较少
