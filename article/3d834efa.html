重装系统后快速恢复Hexo环境

发表于 2022-05-29 更新于 2022-07-26 分类于 经验 阅读次数： Waline： 本文字数： 641 阅读时长 ≈ 1 分钟

本方案仅适用于原项目文件夹未被删除的情况，数据无价，做好备份。

一、安装Git与Node.js

Git for Windows: https://git-scm.com/
Node.js: https://nodejs.org/en/

二、生成SSH密钥

git config --global user.name "这里输入在GitHub的账户名"
git config --global user.email "这里输入在GitHub的注册邮箱名"
ssh-keygen -t rsa -C "这里输入在 GitHub 的注册邮箱"

一路回车即可。

三、设置Github的SSH密钥

C:\Users\主用户\.ssh里有id_rsa和id_rsa.pub两个文件，复制id_rsa.pub的内容。

登录 GitHub 账户，点击右上角个人头像的下拉菜单，依次点击 Settings > SSH and GPG Keys，在 SSH Keys 页面右上角有个New SSH Key按钮，点击该按钮后，粘贴上刚才复制的公钥内容。这样，SSH Key 就配置好了。

四、安装Hexo与相关依赖

打开项目根目录，Git Bash输入命令

npm install hexo-cli -g
npm install
npm install hexo-deployer-git --save

五、验证

hexo clean && hexo g && hexo s
hexo clean && hexo g && hexo d

如果hexo命令不可用，需要修改系统环境配置。
